# Job Tracker Pro

A web application to track job applications, interviews, and follow-ups in one place.

## Features
- Add and manage job applications
- Application status tracking
- Notes and reminders
- Responsive UI

## Tech Stack
- React (Vite)
- TypeScript
- Tailwind CSS
- Node.js / Express
- MongoDB

## Why I built this
While applying for internships and jobs, I found it hard to track multiple applications.  
This project solves that problem by keeping everything organized in one dashboard.

## Setup
npm install
npm run dev
